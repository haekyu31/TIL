### clone, pull ###
1. git clone
    > github에 있는 저장소에서 커밋내역을 가져와 로컬에 저장하는 방식
    로컬 컴퓨터에 원격저장소를 복제하여 로컬저장소를 만든다

2. git pull
   >원격저장소 커밋을 가져오는 형태 로컬저장소의 커밋을 원격저장소와 동기화 하고싶을때 사용한다.
